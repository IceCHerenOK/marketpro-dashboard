# Инструкция по установке и запуску

## Быстрый старт

### 1. Установка Node.js
Убедитесь, что у вас установлен Node.js версии 18 или выше:
```bash
node --version
npm --version
```

Если Node.js не установлен, скачайте с официального сайта: https://nodejs.org/

### 2. Установка зависимостей
```bash
npm install
```

### 3. Настройка переменных окружения
Файл `.env` уже создан с базовыми настройками. При необходимости добавьте ваши API ключи маркетплейсов.

### 4. Запуск приложения

#### Веб-версия (рекомендуется для разработки)
```bash
npm run dev
```
Откройте браузер и перейдите по адресу: http://localhost:5173

#### Десктопная версия
```bash
npm run electron:dev
```

## Сборка для продакшена

### Веб-версия
```bash
npm run build
```

### Десктопное приложение для Windows
```bash
npm run dist:win
```
Исполняемый файл будет создан в папке `release/`

## Возможные проблемы и решения

### Ошибка "PowerShell execution policy"
Запустите PowerShell от имени администратора и выполните:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Ошибка "Port already in use"
Если порт 5173 занят, Vite автоматически выберет другой порт (например, 5174).

### Ошибки TypeScript
Убедитесь, что все зависимости установлены:
```bash
npm install --save-dev @types/compression
```

### Проблемы с SQLite
На Windows может потребоваться установка Visual Studio Build Tools:
```bash
npm install --global windows-build-tools
```

## Структура проекта

```
├── src/
│   ├── frontend/          # React приложение
│   │   ├── components/    # Компоненты
│   │   ├── pages/        # Страницы
│   │   ├── contexts/     # React контексты
│   │   └── utils/        # Утилиты
│   ├── backend/          # Node.js сервер
│   │   ├── routes/       # API роуты
│   │   ├── services/     # Сервисы маркетплейсов
│   │   ├── middleware/   # Middleware
│   │   └── database/     # База данных
│   └── electron/         # Electron приложение
├── dist/                 # Собранные файлы
├── release/             # Исполняемые файлы
└── data/                # База данных SQLite
```

## Первый запуск

1. Запустите приложение: `npm run dev`
2. Откройте браузер: http://localhost:5173
3. Зарегистрируйтесь или войдите в систему
4. Перейдите в раздел "Маркетплейсы" и добавьте API ключи
5. Запустите синхронизацию данных

## Поддержка

При возникновении проблем проверьте:
- Версию Node.js (должна быть 18+)
- Установлены ли все зависимости
- Правильность API ключей
- Логи в консоли браузера и терминале
